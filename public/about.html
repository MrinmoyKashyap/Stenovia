About Stenovia

Stenovia is a lightweight typing-speed platform built with nothing more than HTML, vanilla JavaScript, plain CSS and a minimal Node + Express + MySQL API. Our goal is to prove that you don’t need heavy frameworks to craft fast and beautiful web experiences.

Features

  • 30-second typing test with real-time WPM & accuracy
  • User accounts, JWT auth, personal history
  • Global leaderboard (top average WPM)
  • Fully responsive, dark-mode design
